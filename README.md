## 🛠️ En qué estoy trabajando

* 🌊 **PWA Social para Club:** Desarrollando una Progressive Web App para mi club náutico. Integra APIs en tiempo real para la altura del río y el clima, además de ofrecer una plataforma social y de información para los socios. Mi objetivo es que sea la herramienta definitiva para la comunidad del club.
* 📈 **Ecosistema Vortex:** Mantengo y sigo evolucionando **Vortex Pulse** (oscilador) y **Vortex Prime** (indicador). Son herramientas personales de análisis técnico que uso para experimentar con visualización de datos complejos.
